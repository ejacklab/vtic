# Verification Recipes

## Code / Config Changes

Verify with:
- tests
- lint/type checks if available
- direct behavior checks
- diff review for unintended changes

## Research / Analysis

Verify with:
- sources checked
- summary matches evidence
- uncertainties called out explicitly

## Workflow / Harness Changes

Verify with:
- files exist where expected
- AGENTS.md or local hooks reference them correctly
- the new process is simple enough to actually use

## Universal Rule

A step is not complete until there is a real check tied to it.
