# Cleanup

Cleanup happens at the end, never in the middle.

## Rules

- Prefer proposing cleanup before doing it
- Ask before deleting tracked or important files
- `trash` is better than `rm` when possible
- Never touch memory files, diary files, or rules casually

## Safe cleanup examples
- caches
- temp files
- obvious generated leftovers

## Unsafe cleanup examples
- docs someone may still need
- task/state/history files
- anything with uncertain value
