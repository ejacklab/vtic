**New release: v2026.3.23 (today!)**

- **Channel auth fix** — Auto-select single configured login-capable channel, hardened channel ids against prototype-chain abuse
- **Auth token fix** — Stop live gateway auth-profile writes from reverting saved credentials, token-paste flows now work correctly
- **Browser/MCP fixes** — Chrome MCP wait for usable tabs after attach (reduces macOS consent churn), CDP reuses running browser after brief reachability miss (fixes second-run regressions)
- **Web search fix** — Now uses active runtime provider instead of stale/default selection
