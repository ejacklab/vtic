# Daily Tech Radar

This folder contains daily reports on technology updates relevant to our stack.

## Tracked Projects

| Project | URL | Purpose |
|---------|-----|---------|
| OpenClaw | https://github.com/openclaw/openclaw | Platform we run on |
| ZAI/GLM-5 | https://github.com/zai-org | Primary LLM provider |
| Gemini CLI | https://github.com/google-gemini/gemini-cli | Google's CLI tool |

## Schedule

**Every day at 4:00 PM (GMT+8)** — Cron job `cclow-daily-tech-radar`

## Report Format

Each daily report includes:
- New releases/versions
- Notable commits/features
- Breaking changes
- Action items (if any)
